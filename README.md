# Minimax
Minimax adalah algoritma yang dapat menentukan hasil dari sebuah permainan kompetitif 2 pemain jika kedua pemain bermain secara optimal. Pada kecerdasan buatan, algoritma Minimax bisa digunakan untuk menentukan strategi optimal untuk memenangkan permainan. Minimax akan mengevaluasi secara rekursif dengan melakukan traversal pada pohon game state. Pada setiap langkah, jika langkah ini merupakan langkah kita, maka kembalikan child state dengan nilai tertinggi (state terbaik untuk kita). Sedangkan jika langkah ini merupakan langkah musuh, maka kembalikan child state dengan nilai terendah (state terburuk untuk kita). Algoritma ini dapat diaplikasikan pada permainan Tic-Tac-Toe, misalnya dengan memberi nilai 1, 10, dan 100 untuk 1, 2, dan 3 simbol sama yang berurutan (positif jika simbol kita dan negatif kalau simbol musuh).
## Alpha-Beta Pruning
Alpha-Beta pruning adalah teknik optimisasi yang dapat digunakan pada Minimax. Idenya adalah, jika sudah ada nilai terendah/tertinggi yang bisa dijamin pada suatu state, maka tidak perlu lagi mengevaluasi sebagian child state. Alpha adalah nilai tertinggi yang bisa dijamin pada giliran kita, sedangkan Beta nilai terendah yang bisa dijamin pada giliran musuh. Pada awalnya, Alpha di-set negatif tak hingga dan Beta di-set tak hingga. Nilai ini kemudian diturunkan kepada setiap state yang dilalui kecuali leaf atau end game state yang memiliki nilai sesuai nilai state tersebut. Pada setiap langkah, jika langkah ini merupakan langkah kita, maka nilai Alpha akan di-set menjadi yang terbesar antara nilai Alpha saat ini dengan nilai child state yang dievaluasi. Sedangkan jika ini merupakan langkah musuh, maka nilai Beta akan di-set menjadi yang terkecil antara nilai Beta saat ini dengan nilai child state yang dievaluasi. Jika pada suatu node nilai Alpha sudah lebih besar atau sama dengan Beta, maka terjadi pruning, yaitu langsung mengembalikan nilai ke state yang lebih tinggi tanpa memedulikan nilai child state yang belum dievaluasi.
